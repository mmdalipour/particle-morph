How to Run Examples

Each example is a standalone Next.js page component. To run an example:

1. Copy the code from the desired example's page.tsx file
2. Replace the content in app/page.tsx with the copied code
3. Run npm run dev or pnpm dev
4. Open http://localhost:3000 in your browser

For more detailed documentation, see the README.md file in the examples directory.
